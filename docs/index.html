Colours and Styles

The full set of color options for ev classes are:

  • ev
  • ev-green
  • ev-red
  • ev-yellow
  • ev-green
  • ev-blue
These apply to buttons, windows, and any other UI element. They can also be combined with:
  • ev-glass
  • ev-lg
  • ev-presentation
Examples below.

<button class="ev ev-lg ev-green">Button</button> <button class="ev ev-lg ev-blue ev-glass">Glass Button</button> <button class="ev ev-lg ev-yellow ev-glass ev-presentation">Presentation Style Glass Button</button>
